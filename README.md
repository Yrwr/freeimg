# 免费图床
该免费图床改自imgurl，是一款简单、纯粹的图床程序，使用PHP + SQLite 3开发。

### 主要功能
- [x] 支持拽拖上传、多图上传、Ctrl + V粘贴上传、URL上传
- [x] 支持图片裁剪，自动生成缩略图
- [x] 限制访客上传数量
- [x] 图片压缩/批量压缩
- [x] 图片鉴黄/批量鉴黄

### 环境要求
* PHP >= 5.6
* PDO_SQLite
* GD2
* ImageMagick
* fileinfo
* pathinfo
